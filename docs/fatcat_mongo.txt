Here is the data structure of the mongoose database.

📂 `users`
    _id: ObjectId
    telegramId: int
    username: str
    displayName: str
    firstName: str
    chatTimeToday: int (seconds)
    lastChatActivity: ISO datetime
    lastDailyBonus: ISO datetime
    twitterId: str
    twitterusername: str
    wallets:
        solana: str (wallet address)
        evm: str (optional, wallet address)
    groupPoints:
        [groupId: string]:
            points: float
            invites: int
            messageCount: int
            raids: int
    groupInvites:
        [groupId: string]:
            invitedUsers: array of Telegram IDs
            groups: array of groupId ints
    projects: array of project names (e.g. "Fat Cat - $FATCAT")

❗ Do not expose:
twitterKeysV2 (contains accessToken, refreshToken, lastRefresh — sensitive OAuth credentials)

📂 `projects`
- name: str
- displayName: str
- groupId: int
- telegramId: str
- inviteLink: str
- stats: { totalPoints: float, memberCount: int }

📂 `raids`
- groupId: int
- status: 'in_progress' | 'completed'
- tweetUrl: str
- tweetContent: { text: str, author: { username: str }, media: [] }
- createdAt: datetime
- duration: int
- requiredActions: { like, retweet, reply, bookmark: bool }
- statistics: { likes, retweets, replies, bookmarks: int }
- participants: array of users

📂 `group_subs`
- groupId: int
- tier: str
- subscribedAt: ISO datetime
- renewalDate: ISO datetime
